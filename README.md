# Sharmaine Perea — Software Engineer Portfolio

Personal portfolio website for Sharmaine Perea, a software engineer with a background in professional ballet and small business ownership.

**Live site:** [Sharmaine Perea] (https://sharmaine-perea.vercel.app/)

---

## Features

An editorial-style single-page portfolio built in React + Vite, featuring my signature slate-and-gold design. Sections include:

- **About Me** — BFA in Dance from UMKC, 10 years with San Diego Ballet, founder of Charms Dancewear, and Flatiron School graduate
- **Brand** — Charms Dancewear, a custom hand-crafted dancewear business (active on Etsy)
- **Development** — Skills and approach to building software
- **Projects** — featured work including Stitch Flow (full-stack sewing project manager), Paradox Golf, Breakaway Trivia, and Lunaluxe
- **Contact** - GitHub and LinkedIn

## Tools
- IDE (VS Code)
- Git + GitHub
- React (Vite)
- CSS 
- Deployed on Vercel

## Run Locally

```bash
git clone https://github.com/webdesigns23/software-engineer-portfolio-website
cd software-engineer-portfolio-website
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).
